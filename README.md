# Project-1

The purpose of this project is to evaluate math expression.

# Features:

 # Basic Arithmetic Operations:

 Addition (+)
 Subtraction (-)
 Multiplication (×)
 Division (÷)

 # Advanced Math Functions:

 Exponentiation (^)
 Square root (√)
 Logarithms (log, ln)
 Trigonometric functions (sin, cos, tan, cot)

 # Constants:

 π (Pi)
 e (Euler's number)

 # Backspace:

 Allows the user to delete the last entered character or operation.

 # Clear Button:

 Resets the display to start a new calculation.
